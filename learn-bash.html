Overview of Bash

Bash is a Unix shell and command language written by Brian Fox for the GNU Project as a free software replacement for the Bourne shell. First released in 1989, it has been used as the default login shell for most Linux distributions. Bash was also the default shell in all versions of Apple macOS prior to the 2019 release of macOS Catalina, which changed the default shell to zsh, although Bash currently remains available as an alternative shell

Tips & Terminology

Directory – Folder

“Exit” Command  – Exits the connected session

“–help”  Lists other attributes that can be used (eg. “ls –help)

Shell – Command language used (Bash)

Linux – Operating system

IP Address – Numerical label assigned to each device or server

Port – Communication endpoint

Connecting to a server

When connecting to a server, SSH (Secure Shell Protocol) is used on port 22. There are many different attributes that can be used with the SSH command. 

				
					ssh USERNAME@IP-ADDRESS -p PORT #Connects to an IP address at a specfic port as a user

List Conents

If a user wants to see the contents of the current directory they can use the “ls” command.

				
					ls #List all unhidden content in the current directory

ls -a #List all hidden and unhidden content in the current directory

ls /home/ #List all unhidden conent in the "home" directory

Run commands together

We can also run two commands at the same time, this lets us be more efficient. 

				
					cd /videos/ && ls # Changes to the videos directory and lists the contents

Change Directory

If we need to navigate to a different directory we can use the “cd” command.

				
					cd /videos/ #Change to the videos directory

cd -  #Revert back one directory

cd ~ #Change to the home directory

Create file/directory

Sometimes we might need to create a new file or directory. 

				
					touch MyFile.txt #Creates a text file called MyFile

mkdir NewPhotos #Creates a new directory called NewPhotos

Edit, Copy or Open File

After creating a file we might need to modify the contents or copy the file.

				
					nano MyFile.txt #Opens MyFile.txt in a word editor

cat MyFile.txt #Prints the contents of MyFile.txt

cp MyFile.txt NewName.txt #Copies the MyFile.txt and created a new file called NewName.txt

Find contents

We may also need to find a spefic item within a file or folder.

				
					grep Bill Accounts.txt #Prints all the words with "Bill" in the file

tail -n 5  MyFile.txt #Prints the last five lines of the file (-n 5 = only the last five lines)

Zip/Unzip

If we are sending or looking to reduce the file size we may need to compress a directory or file.

				
					zip Zipfile MyFile #Compresss the MyFile and creates a MyFile.zip

unzip ZipFile.zip # Decompresses the contents of the ZipFile.zip
